WPS Menu Test

This tests the WPS.Menu component (and the WPS.Info component) from the WPS.Ui.js module. Selecting an algorithm (aka offering) from the Algorithms list will describe that algorithm in the Metadata WPS.Info box, on the right hand side of the page. Furthermore, the selected algorithm will be executed, with hard-coded parameters, and the raw output displayed in the Data WPS.Info box, in the centre of the page.
