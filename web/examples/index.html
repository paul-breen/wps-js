WPS Examples

The following are a number of example pages that illustrate the WPS Javascript Library. The pages request meteorological data from British Antarctic Survey, via calls to the WPS at http://sosmet.nerc-bas.ac.uk:8080/wpsmet/WebProcessingService (default WPS test client).

    • wps-core-test.html
  • WPS user interface components
    • wps-menu-test.html
    • wps-info-test.html
  • WPS integration with SOS.Ui components
    • wps-met-monthly-means-plot-test.html
